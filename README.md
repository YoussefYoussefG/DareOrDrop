# Dare or Drop

Dare or Drop is a premium memory game built with React Native and Expo. It features a modern dark-mode aesthetic with neon accents, smooth animations, and cross-platform support.

## Getting Started

To run the application locally:

1. Make sure you have Node.js installed.
2. Navigate to this directory in your terminal:
   ```bash
   cd "DareOrDrop"
   ```
3. Install dependencies (if you haven't already):
   ```bash
   npm install
   ```
4. Start the Expo development server:
   ```bash
   npx expo start
   ```

### Running on Different Platforms:
- **Web**: Press `w` in the terminal to open the game in your web browser.
- **Android**: Press `a` to open in an Android Emulator.
- **iOS**: Press `i` to open in an iOS Simulator (requires macOS).
- **Physical Device**: Download the "Expo Go" app on your iOS or Android device and scan the QR code displayed in your terminal.

## Features
- Interactive 4-color memory pads (Red, Blue, Green, Yellow).
- Smooth 60fps micro-animations using `react-native-reanimated`.
- Haptic feedback on physical devices using `expo-haptics`.
- Responsive layout that works on mobile phones, tablets, and web browsers.
